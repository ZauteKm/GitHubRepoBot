# Message-Search-Bot

A Telegram Bot for searching any channel messages from Inline by [@ZauteKm](https://github.com/ZauteKm).

I made this for [@BotzListBot](https://t.me/BotzListBot). You can use this for something else. Edit according to your use.

We have to use Bot for Inline Search & Userbot for Searching in Channels. So both Bot & Userbot will work together.
